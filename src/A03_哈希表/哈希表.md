# 哈希表


## Java哈希表用法
### merge用法
1. merge(K key, V value, BiFunction<? super V, ? super V, ? extends V> remappingFunction)
+ key：要操作的key
+ value：要插入或者合并的值
+ remappingFunction：key存在时的合并函数（接收旧值和新值，返回新值）

总结：merge = 不存在则创建 + 存在则按规则合并

``` java
Map<String, Integer> map = new HashMap<>();

// 场景 2-1：取最大值（比如记录最高分）
map.merge("player1", 80, Math::max);  // 不存在，放入 80
map.merge("player1", 95, Math::max);  // 已存在，Math.max(80, 95) = 95
map.merge("player1", 70, Math::max);  // 已存在，Math.max(95, 70) = 95（保持不变）
System.out.println(map.get("player1")); // 95

// 场景 2-2：取最小值（比如记录最短时间）
map.merge("level1", 100, Math::min);  // 放入 100
map.merge("level1", 85, Math::min);   // Math.min(100, 85) = 85
System.out.println(map.get("level1")); // 85

// 场景 2-3：累加但有上限
map.merge("energy", 50, (oldVal, newVal) -> Math.min(oldVal + newVal, 100));
map.merge("energy", 60, (oldVal, newVal) -> Math.min(oldVal + newVal, 100));
System.out.println(map.get("energy")); // 100（不超过上限）

// 场景 2-4：字符串拼接
Map<String, String> tags = new HashMap<>();
tags.merge("doc1", "java", (oldVal, newVal) -> oldVal + "," + newVal);
tags.merge("doc1", "spring", (oldVal, newVal) -> oldVal + "," + newVal);
System.out.println(tags.get("doc1")); // "java,spring"

```


```java
Map<String, Integer> map = new HashMap<>();
map.put("a", 5);
map.put("b", 3);

// 场景 3-1：直接删除
map.merge("a", 0, (oldVal, newVal) -> null);
System.out.println(map.containsKey("a")); // false
System.out.println(map.size());           // 1（只剩 b）

// 场景 3-2：条件删除（比如值为 0 时删除）
map.put("x", 0);
map.put("y", 10);
map.merge("x", 0, (oldVal, newVal) -> oldVal == 0 ? null : oldVal + newVal);
map.merge("y", 0, (oldVal, newVal) -> oldVal == 0 ? null : oldVal + newVal);
System.out.println(map); // {y=10}，x 被删除

// 场景 3-3：计数器归零时自动删除
Map<String, Integer> countMap = new HashMap<>();
countMap.put("item1", 3);

// 减少计数
countMap.merge("item1", 1, (oldVal, delta) -> {
    int newVal = oldVal - delta;
    return newVal == 0 ? null : newVal;  // 为 0 时删除
});
System.out.println(countMap); // {item1=2}

countMap.merge("item1", 2, (oldVal, delta) -> {
    int newVal = oldVal - delta;
    return newVal == 0 ? null : newVal;
});
System.out.println(countMap); // {}，item1 被删除

```
